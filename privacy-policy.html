Privacy Policy for Stock Market Analysis Extension

Last Updated: December 20, 2024

Overview

Stock Market Analysis ("we", "our", or "the extension") is committed to protecting your privacy. This privacy policy explains how our Chrome extension collects, uses, and safeguards information.

Information Collection

Data We Collect

Our extension collects minimal data to provide stock market analysis functionality:

  1. Stock Symbol Queries: When you search for or click on stock symbols, we temporarily process these queries to fetch real-time market data.
  2. No Personal Information: We do NOT collect:
    • Personal identification information
    • Browsing history
    • Email addresses
    • Payment information
    • Location data
    • Cookies or tracking data

How We Use Information

The stock symbols you interact with are used solely to:

  • Fetch real-time stock prices from Yahoo Finance API
  • Retrieve currency exchange rates
  • Generate AI-powered analysis
  • Display technical indicators

All data processing happens locally in your browser or through public APIs. We do not store or transmit your personal data to our servers.

Third-Party Services

Our extension uses the following third-party services:

  1. Yahoo Finance API
    • Purpose: Real-time stock quotes and market data
    • Data Shared: Stock symbols only
    • Privacy Policy: Yahoo Privacy Policy
  2. Exchange Rate API
    • Purpose: Currency conversion (USD to KRW)
    • Data Shared: None (public API)
    • Privacy Policy: Exchange Rate API Terms
  3. grcbt.com
    • Purpose: Detailed stock analysis (optional, user-initiated)
    • Data Shared: Stock symbol via URL parameter when user clicks link
    • Privacy Policy: grcbt.com Privacy Policy

Permissions Explanation

Our extension requests the following Chrome permissions:

  1. storage: To cache stock data temporarily (improves performance, reduces API calls)
  2. activeTab: To detect stock symbols on the current webpage you're viewing
  3. host_permissions: To access Yahoo Finance and Exchange Rate APIs for fetching stock data

We use the minimum permissions necessary for functionality.

Data Storage

  • Local Storage Only: Stock data is cached locally in your browser for 1-60 minutes to improve performance
  • No Cloud Storage: We do not store your data on external servers
  • Automatic Deletion: Cached data is automatically cleared when cache expires

Data Sharing

We do NOT:

  • Sell your data to third parties
  • Share your data with advertisers
  • Track your browsing behavior
  • Create user profiles

User Control

You have full control over the extension:

  • Disable anytime: Remove the extension from Chrome to stop all data processing
  • Clear cache: Clear browser cache to remove stored stock data
  • No account required: The extension works without registration or login

Children's Privacy

Our extension is not directed to children under 13. We do not knowingly collect information from children.

Changes to Privacy Policy

We may update this privacy policy occasionally. Changes will be posted with a new "Last Updated" date.

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)

Your Rights

You have the right to:

  • Know what data is collected (minimal, as described above)
  • Request data deletion (uninstall the extension)
  • Opt-out of data collection (don't use the extension)
